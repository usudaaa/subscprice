## 価格改定の記入方法

価格が変わったときは **行を追加** します。古い行は消さずに残してください。

### 例：月額 1,000円 → 1,200円 に値上げ（2025-06-01〜）

**変更前**

| プラン | プラン説明 | 適用開始 | 適用終了 | 価格 | 課金 | 税 | 備考 |
|---|---|---|---|---|---|---|---|
| 個人 |  | 2023-01-01 |  | 1,000円 | 月額 | 税込 | 旧価格 |

**変更後**

| プラン | プラン説明 | 適用開始 | 適用終了 | 価格 | 課金 | 税 | 備考 |
|---|---|---|---|---|---|---|---|
| 個人 |  | 2023-01-01 | 2025-05-31 | 1,000円 | 月額 | 税込 | 旧価格 |
| 個人 |  | 2025-06-01 |  | 1,200円 | 月額 | 税込 | 価格改定 |

ポイント：
- **適用終了** は値上げ前日の日付を入力（YYYY-MM-DD形式）
- 新しい行の **適用終了** は空欄のまま（＝現在価格）
- 日付は `2025-06-01` 形式（ハイフン区切り）

---

## 各列の意味

| 列名 | 内容 | 例 |
|---|---|---|
| プラン | プラン名 | 個人、ファミリー、Pro |
| プラン説明 | プランの補足説明 | 6人まで、50GBストレージ込み |
| 適用開始 | その価格の開始日 | 2023-06-01 |
| 適用終了 | その価格の終了日。現在価格は **空欄** | 2025-05-31 |
| 価格 | 金額（円またはUSD） | 1,200円 / 20 USD |
| 課金 | 月額 or 年額 | 月額 |
| 税 | 税込 or 税別 | 税込 |
| 備考 | 改定の理由など | 価格改定、サービス開始時 |

## 修正後の反映手順

編集が完了したら Claude Code に以下のように依頼してください：

> 「data/md/ のMarkdownファイルを読み込んで、index.html と edit.html のDBを更新してください」

Claude が自動でデータを解析し、HTMLファイルのインラインJSONを更新します。
