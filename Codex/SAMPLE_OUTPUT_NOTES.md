| 適用期間              | 価格              | 備考         |

## 3. 動作確認シナリオ

1. **index.html をダブルクリック** → ブラウザで開く（`file://` で動作）
2. カテゴリ「AI」をクリック → ChatGPT, Claude, Gemini の 3 サービスのみ表示
3. Claude の「詳細・推移」ボタン → モーダルで Pro/Max 5x/Max 20x が表示される
4. モーダル右上の × またはオーバーレイクリック → 閉じる
5. Escape キー → 閉じる
6. **edit.html をダブルクリック** → 編集ツールが開く
7. サイドバーで DAZN を選択 → フォームに値が入る
8. 月額を 4,200 → 4,800 に編集して「保存」
9. 「全データをコピー／エクスポート」 → 出力に `4800` が入っている
10. `index.html` の `const DB = ...` を貼り替え → 反映される

## 4. よくある実装ミス

- `fetch()` を使ってしまい `file://` で CORS エラー → 必ずインライン JSON にする
- 改定履歴の並び順を間違える（古い順／新しい順）→ モーダルでは「新しい順」表示が読みやすい
- USD の税表記欠落 → `（税別）` を必ず付ける
- 日本語 fonts 指定がなくシステムフォント → Hiragino Sans 系のフォールバック必須
- カテゴリ順がアルファベット順になる → 仕様の順序を必ず守る
- USD の表示色が同じ → 緑系（`#1a6e40`）で区別すると読みやすい
