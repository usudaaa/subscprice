# 開発依頼プロンプト

Impress Watch（インプレスウォッチ）向けの **「サブスクリプション価格ウォッチ」** という単一ページ Web コンテンツを、純粋な静的 HTML/CSS/JavaScript のみで作ってください。フレームワークもビルドツールも使わず、ファイルを直接ブラウザで開いて動作することが必須です（`file://` でも動作すること）。

## 成果物

以下 2 ファイルをプロジェクトルートに生成してください：

1. **`index.html`** — 一般読者向けの公開ページ
2. **`edit.html`** — 編集スタッフ向けの内部 GUI（同じデータを GUI で編集できるツール）

データは外部 JSON を `fetch()` せず、各 HTML 内に **`const DB = {...}`（index.html）** または **`let DB = {...}`（edit.html）** としてインライン記述してください。CORS 制約を回避するためです。

## 投入データ

添付の **`data.json`** を両ファイルに同じ内容で埋め込んでください。

## 詳細仕様

- 機能・UI 仕様: **`SPEC.md`** を参照
- デザイン仕様: **`DESIGN.md`** を参照（Impress Watch のネイビーブルー基調）

## 重要なポイント（要約）

- Impress Watch の既存サイト（https://www.watch.impress.co.jp/）と同系統の落ち着いた配色・レイアウト
- メインカラーは **ネイビーブルー `#003399`**（赤系は使わない）
- データ更新頻度は低い（数年に 1 回程度）—— 自動取得は不要、手動編集ベース
- カテゴリ別にサービスを整理し、現在価格を一覧表示
- 各サービスの「詳細」ボタンを押すとモーダルが開き、**過去の価格改定履歴**が表で見られる
- 価格は **税込 JPY**（円表示）または **税別 USD**（米ドル表示、月額のみ）
- `edit.html` では GUI でデータを編集 → 「全データをコピー／エクスポート」で JSON を取り出せる

## まず最初に

`index.html` を完成させてから `edit.html` に進んでください。完成後、両ファイルがブラウザで開けることを必ず確認してください。
