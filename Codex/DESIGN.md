# デザイン仕様 — Impress Watch トンマナ

参考: https://www.watch.impress.co.jp/

## 1. カラーパレット

```css
:root {
  --iw-navy:       #003399;  /* メインカラー：ヘッダー、見出し背景、強調 */
  --iw-blue:       #0055cc;  /* ボタン、リンク強調 */
  --iw-blue-mid:   #1a5fb4;  /* モーダル見出し背景 */
  --iw-blue-light: #e8f0fb;  /* ページヘッダー背景、ホバー */
  --iw-dark:       #1a1a1a;  /* 本文テキスト */
  --iw-gray:       #555555;  /* 補助テキスト */
  --iw-light-gray: #f4f6f9;  /* セクション背景 */
  --iw-border:     #d0d8e8;  /* 表・カード罫線 */
  --iw-link:       #0044bb;  /* テキストリンク */
  --iw-link-hover: #cc2200;  /* リンクホバー（赤系アクセント） */
}
```

**重要**: メインカラーはネイビーブルー `#003399`。**赤系をベースに使わない**こと（赤は控えめなアクセントのみ）。

## 2. タイポグラフィ

```css
--font-body: "Hiragino Sans", "Hiragino Kaku Gothic ProN", "Noto Sans JP", Meiryo, sans-serif;
```

- 本文サイズ: **14px**
- 行高: **1.7**
- 見出し: ボールド、ネイビーまたは白抜き

## 3. レイアウト

- コンテンツ幅: **最大 980px**（中央寄せ）
- パディング: 左右 12px

## 4. 主要パーツのスタイル

### サイトヘッダー
- 背景: `--iw-navy`
- 高さ: 48px
- 左に「Impress Watch」風ロゴ（白文字）、右にパンくず（うす白）

### ページヘッダー
- 背景: `--iw-blue-light`
- 下境界: `--iw-navy` 2px solid
- タイトル: 22px ボールド `--iw-navy`、左に 4px の縦バー

### カテゴリナビ（タブ）
- 横並びボタン、`border: 1px solid #b0bcd8`、`border-radius: 3px`
- 通常: 白背景、`color: --iw-dark`
- ホバー: 背景 `--iw-blue-light`、枠 `--iw-blue`
- アクティブ: 背景 `--iw-navy`、文字白

### カテゴリ見出し
- 背景: `--iw-navy`
- 文字: 白、ボールド 15px
- パディング: 7px 14px、上端のみ角丸 3px

### サービス一覧テーブル
- 罫線: `1px solid --iw-border`
- ヘッダー行: 背景 `#dce6f5`、文字 `--iw-gray` 12px
- 行ホバー: 背景 `--iw-blue-light`
- 列幅: 22% / 60% / 18%

### 「改定履歴あり」バッジ
- 背景: `#fff8dc`、枠 `1px solid #c8a000`
- 文字: `#7a5c00`、10px

### 「詳細・推移」ボタン
- 背景: `--iw-blue`、文字白、`border-radius: 3px`
- ホバー: 背景 `--iw-navy`

### モーダル
- オーバーレイ: `rgba(0,0,0,.55)`
- ボックス: 白、`max-width: 720px`、`border-radius: 4px`
- ヘッダー: 背景 `--iw-navy`、文字白
- セクション見出し: 背景 `--iw-blue-mid`、文字白、`border-radius: 2px`
- アニメーション: 0.18s スライドイン

### 価格履歴テーブル（モーダル内）
- ヘッダー: 背景 `#eef2f7`、文字 `--iw-gray`
- 「現在」バッジ: 背景 `--iw-navy`、文字白
- 「値上げ」バッジ: 背景 `#fdecea`、文字 `#c0392b`
- 「値下げ」バッジ: 背景 `#eafaf1`、文字 `#1a6e40`

## 5. 価格表示の配色

- 円表示: `--iw-dark`（通常の濃グレー）
- USD 表示: `#1a6e40`（やや暗めの緑、税別であることを視覚的に区別）

## 6. レスポンシブ

- スマホ表示（〜640px）: テーブルはそのまま縦スクロール、モーダルは横幅 `calc(100% - 24px)` 程度
- 横スクロールが出ても OK（深い対応は不要）

## 7. アクセント・装飾の指針

- 影は最小限（モーダルのドロップシャドウのみ）
- アニメーションは控えめ（モーダル開閉のみ）
- 角丸は 3〜4px と小さめ（カチッとした業界誌風）
- 過度な装飾より「読みやすさ」「整理感」を優先
