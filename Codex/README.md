# Codex 用 開発素材

別のコーディングエージェント（Codex 等）に同じプロジェクトを作らせるための素材一式です。

## ファイル構成

| ファイル | 用途 |
|---|---|
| `PROMPT.md` | エージェントに投げる**メインプロンプト**（最初に渡す） |
| `SPEC.md` | 詳細仕様書（UI・データスキーマ・機能要件） |
| `DESIGN.md` | デザイン仕様（Impress Watch トンマナ、配色、レイアウト） |
| `data.json` | 投入する全データ（カテゴリ・サービス・価格履歴） |
| `SAMPLE_OUTPUT_NOTES.md` | 出力例・確認ポイント |

## 使い方

1. **PROMPT.md** をコピーしてエージェントに貼り付け
2. 必要に応じて **SPEC.md** / **DESIGN.md** / **data.json** を補足資料として渡す
3. エージェントが生成した `index.html` と `edit.html` を動作確認
